# Code of Conduct

Outpost follows the [Contributor Covenant](https://www.contributor-covenant.org/),
version 2.1. The full text is available at:

  https://www.contributor-covenant.org/version/2/1/code_of_conduct/

In short:

- Treat everyone with respect.
- No harassment, discrimination, or personal attacks.
- Keep discussion focused on the project and its goals.
- Disagree about the work, not about the person.

Maintainers may remove, edit, or reject contributions and ban contributors
for violations. Report concerns privately via the contact in
[`SECURITY.md`](SECURITY.md).
